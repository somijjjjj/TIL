# No.2739 _ 구구단
- N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램
- 입력 N (1 <= N >= 9)
- 출력형식과 같게 N * 1 부터 N * 9 까지

### 1. 내가 작성한 코드
```
n = int(input())

for i in range(1,10):
    print('{0} * {1} = {2:<2}'.format(n, i, n*i))
```
- 입력받은 N의 구구단을 출력하기 위해 1~9 사이의 반복문 생성
- for문 실행으로 n단의 1~9까지 구구단 출력
- [format 함수 사용](python\format 함수.md)
